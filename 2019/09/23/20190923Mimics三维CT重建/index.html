Mimics三维CT重建

Posted on 2019-09-23 Edited on 2020-05-16 In learning

CT扫描得到的图像是多张dicom格式文件。如果只需要显示图像序列，可以用ImageJ软件导入图像序列查看。Mimics软件（全称Materialise’s Interactive Medical Image Control System）是一个CT、MRI、超声等医学图像的显示和分割工具，可以将断层扫描的结果转换成机械领域软件可以处理的格式，如STL等。

下面以Mimics 21为例，总结最基础的利用多张dcm图片进行三维重建的步骤：

导入文件

打开Mimics软件，点击New Project并导入CT图像。

接下来确认信息，点open

转化完成后一般图像方向也会自动设置好，但有些文件需要手动设置方向，比如下图。一般要从六个方向对图像定位，即上下左右前后，Top、Bottom、Left、Right、Anterior、Posterior。

直接点击“X”进行设置，设置完成后如下图。

图像显示

确定图像方向后，Mimics会自动生成横断面、冠状位、矢状位和三维视图。

界面排版可以在菜单栏VIEW下面的的Layouts处修改。

阈值分割

阈值分割是通过选择一定的像素灰度值范围来对对象进行分割的方法。在CT图像中即为通过设置CT值来分割。

点击菜单栏SEGMENT中的Threshold

在这里设置需要提取的CT值范围。软件中提供了预设的阈值范围，可以先根据感兴趣区域选择一个预设值，然后进行微调。

设置完成后在Project Management中的Masks会出现刚刚提取的区域。

三维显示及导出

点击当前Mask，右键菜单栏选择Calculate Part，Calculate。

等待一段时间后，Objects栏会出现生成的物体。

最后在Objects栏右键导出STL即可。

STL文件显示如下

# software # Mimics # ImageProcess
