立夏

发表于 2019-05-06 分类于 solar terms Valine：

莫趁垂杨飞絮，且随红药翻阶。
《朝中措》宋·管鉴

《月令七十二候集解》：“立夏，四月节。夏，假也。物至此时皆假大也。 ”在天文学上，立夏表示即将告别春天，是夏天的开始。实际上在我国的大部分地区从平均温度来看并没有真正地进入夏天，而是属于仲春和暮春季节。

这段时间的天气总是稍稍热起来一点就被阵阵凉风压住势头，阳光、微风、绿叶，一切都刚刚好。走在校园的林荫路上，不自觉地会回想起那句“暮春者,春服既成,冠者五六人,童子六七人,浴乎沂,风乎舞雩,咏而归”。高中第一次读到这句话时就忍不住跟着读起来，尤其喜欢最后三句“浴乎沂,风乎舞雩,咏而归”，就像哼唱着一首轻快的歌，心情都愉快起来。

立夏三候：蝼蝈鸣，蚯蚓出，王瓜生。《礼记·月令》篇，解释立夏曰：“蝼蝈鸣，蚯蚓出，王瓜生，苦菜秀。”说明在这时节，蝼蝈开始聒噪着夏日的来临，蚯蚓也忙着帮农民们翻松泥土，乡间田埂的野菜也都彼此争相出土日日攀长。

英文简历注意事项

发表于 2019-04-25 分类于 others Valine：

简历主要部分

个人信息

在英文简历中需要提供的个人信息有：姓名，地址，邮箱，电话。不需要提供的是：年龄，性别，婚姻状态，照片，宗教信仰。

地址：城市后面要写中国，但不必用PRC等，因为用China简单清楚。邮编的标准写法是放在省市名与国名之间，起码放在China之前，因为是中国境内的邮编。

邮箱：尽量用正式邮箱，比如学校邮箱等，不要用qq邮箱等有一大串数字的邮箱。

电话：前面一定加地区号，如（86-10），括号和后面的号码之间加空格。如果是手机号，则要采用“4-3-4”或“3-4-4”格式，如“1380-135-1234”，“138-0135-1234”.

雨生百谷

发表于 2019-04-21 分类于 solar terms Valine：

谷雨是春季最后一个节气。谷雨的到来意味着寒潮天气基本结束，气温回升加快，雨量增多，是万物生长的大好时节。《月令七十二候集解》中说，“三月中，自雨水后，土膏脉动，今又雨其谷于水也……盖谷以此时播种，自下而上也”，谷得雨而生，由此得名。有传说谷雨源于仓颉造字，据《淮南子》记载：仓颉造字，是一件惊天动地的大事，黄帝于春末夏初发布诏令，宣布仓颉造字成功，并号召天下臣民共习之。由于仓颉造字功德感天，玉皇大帝便赐给人间一场谷子雨，以慰劳圣功，这就是现在的“谷雨”节气。

谷雨三候：一候萍始生；二候鸣鸠拂其羽；三候戴胜降于桑。谷雨后雨量增多，浮萍开始生长，接着布谷鸟声声鸣叫，在田野里提醒人们要开始播种，进而桑树上开始见到戴胜鸟，是蚕宝宝将要生长的信号。

在气温适宜浮云蔽日的天气里登上紫金山，站在山顶放眼望去一片翠绿，令人心旷神怡。

早难道好处相逢无一言

发表于 2019-04-19 分类于 journal Valine：

是哪处曾相见，相看俨然，早难道好处相逢无一言

第一次听这句唱词时，我并没有感受到巨大的触动，只是觉得遣词造句给人一种如沐春风般的舒服。如今再回想起这句话，这段故事，心里又多了几分感动和震撼。

人最美好的年华其实不过短短数年。正处在恰好年纪的杜丽娘，每天只能独处深闺，顾影自怜，犹如春意盎然的良辰美景，无人赏识。尽可能地想像古代的文化背景，大抵能体会到这种被压抑被忽视的少女遇到心上人时喷涌而出的炽热感情，真是为情所困、因爱而死、死而复生的至高境界。昆曲的表演是含蓄内敛的，唱到这句词时，生和旦内心恐怕早已按捺不住的喜悦和爱慕，却只是通过水袖翻飞缠绵来表现，连四目相对和指尖的触碰都令二人惊慌失措，含羞转身躲避，只会在背对着对方时流露出窃喜和娇羞。

“相见、相看、相逢”，所有这一切都是两个人相互之间生发的情感，是眉眼躲闪、手指翻飞的心动。该用怎样的语言来形容这种“好”呢？恐怕真的是“无一言”了啊！

大概爱就像藤蔓，野蛮生长紧紧缠绕，让人无法挣脱却又感到踏实而喜悦。

两者都是打开文件用的函数，其中fopen_s是c++11之后增加的安全增强版本，增加了安全检测。

fopen

打开由该文件指示的文件filename并返回指向与该文件关联的文件流的指针。mode用于确定文件访问模式。

函数形式：FILE * fopen (const char * filename, const char * mode);

用法：fp = fopen(filename, "w");
如果打开文件成功，返回文件指针，打开失败则返回NULL。

清明

发表于 2019-04-05 分类于 solar terms Valine：

春城无处不飞花，寒食东风御柳斜。
——《寒食》韩翃

《岁时百问》：“万物生长此时，皆清洁而明净，故谓之清明。”

清明节期在仲春与暮春之交，兼具自然与人文两大内涵，既是自然节气，也是传统节日。清明正值万物生长，春和景明之时，是踏青郊游的快乐节日，因此也被称为踏青节。但同时，自古以来清明也是传统的扫墓祭祀节日。明《帝京景物略》载：三月清明日，男女扫墓，担提尊榼，轿马后挂楮锭，粲粲然满道也。拜者、酹者、哭者、为墓除草添土者，焚楮锭次，以纸钱置坟头。

我从未在清明时扫墓祭奠过，因此并没有亲身体会。提到清明扫墓，我的脑海里总是会想起蔡崇达在《皮囊》中的一段话，当他回故乡祭祖时，那种感受就像自己是大树一圈圈年轮中的一环，“拥挤得那么心安”。这可能就是扫墓祭祖，缅怀先人给予我们的力量吧，它告诉我们从哪里来到哪里去，给我们归属感和认同感，让我们能更加珍惜当下，不负春光。

“一候桐始华；二候田鼠化为鴽；三候虹始见”。在这个时节先是白桐花开放，喜阴的田鼠不见了，全回到了地下的洞中，雨后的天空可以见到彩虹了。

春分

发表于 2019-03-21 分类于 solar terms Valine：

雪入春分省见稀，半开桃李不胜威。
应惭落地梅花识，却作漫天柳絮飞。
不分东君专节物，故将新巧发阴机。
从今造物尤难料，更暖须留御腊衣。
《癸丑春分后雪》宋·苏轼

春分，是春季九十天的中分点，古时又称“日中”、“日夜分”。《月令七十二候集解》：“二月中，分者半也，此当九十日之半，故谓之分。秋同义。”《春秋繁露·阴阳出入上下篇》说：“春分者，阴阳相半也，故昼夜均而寒暑平。”春分也是节日和祭祀庆典，古代帝王有春天祭日，秋天祭月的礼制。民间活动上，一般算做踏青的正式开始。春分是伊朗、土耳其、阿富汗、乌兹别克斯坦等国的新年，有着3000年的历史。

中国古代将春分分为三候：“一候元鸟至；二候雷乃发声；三候始电。”春分日后，燕子便从南方飞来了，下雨时天空开始打雷并发出闪电。

在连续经历了多日艳阳高照后，南京突然迎来了一场雷雨，今天气温就降下来了。本来冬天的厚衣服已经被洗干净收起来，不得不又拿出一件来穿。在这样的情境下读到苏轼的“更暖须留御腊衣”，真是倍感亲切。诗词中不只有宏图远志，也有生活的细枝末节和蕴藏在世间万物的人生智慧。

常量指针

本质是一个指针，常量作为修饰。不能用该指针修改指向的变量。

形式：int const* p const int* p

一雷惊蛰始

发表于 2019-03-06 分类于 solar terms Valine：

微雨众卉新，一雷惊蛰始。
《观田家》韦应物

惊蛰，是二十四节气中的第三个节气，标志着仲春时节的开始。《月令七十二候集解》：“二月节……万物出乎震，震为雷，故曰惊蛰，是蛰虫惊而出走矣。”此前，昆虫入冬藏伏土中，不饮不食，称为“蛰”；到了“惊蛰节”，天上的春雷惊醒蛰居的动物，称为“惊”。故惊蛰时，蛰虫惊醒，天气转暖，渐有春雷，中国大部分地区进入春耕季节。

古代分惊蛰为三候：“一候桃始华；二候仓庚（黄鹂）鸣；三候鹰化为鸠。”

vector内存释放

发表于 2019-02-28 分类于 learning Valine：

vector内存分配特点

vector有一个特点：内存空间只会增长，不会减小。为了支持快速的随机访问，vector容器的元素以连续方式存放，每添加一个元素时，都需要保证连续存放的特性，若空间不足则需要重新分配空间、拷贝元素。因此在创建vector时，对其分配的内存空间比当前需要的要多一些，也就是说预留一些额外的存储空间用于存放新添加的元素。

vector有两个函数，一个是capacity()，返回对象缓冲区（vector维护的内存空间）实际申请的空间大小，另一个size()，返回当前对象缓冲区存储数据的个数。也就是说，前者是占用空间的大小，后者是其中存放数据的多少，并且capacity永远大于等于size。当两者相等时，vector就会重新分配内存，扩大capacity。

对流扩散方程

了解了一维扩散方程的求解原理后，就可以逐渐将其推广至二维、三维以及流动问题。首先加入最简单的流动，本文所指的对流是流体粒子整体运动（平流），即所有粒子的宏观运动速度都相同。

在笛卡尔坐标系中，一维对流扩散方程可以表示为

$$ \frac{\partial \phi }{\partial t}+u\frac{\partial \phi }{\partial x}=\alpha \frac{\partial^2 \phi }{\partial x^2} $$

这个方程描述的是对流和扩散同时发生的过程，比如在恒定速度的水流中滴一滴墨水，墨水浓度随时间变化的情况。
