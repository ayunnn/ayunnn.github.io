沥青的味道

发表于 2018-08-21 分类于 journal Valine：

四牌楼校区最近正在对路面的沥青进行翻修。经过多年的修补，路面早已是补丁摞补丁，确实也该重新铺了。

看到一辆小铲车沿着路边，整整齐齐地铲起一块块沥青，我脑海里浮现出许多年前，在奶奶家围观铺路的情景。村子里的主干道是一条土路，每逢夏季暴雨或秋季连阴雨，就会变得相当泥泞，难以行走。因此大家迫切地希望有一条干净硬实的沥青马路。

那天，我和其他好奇的围观者一起，早早地站在路边，从路面压平，倒上粘稠的沥青，再到用宽大滚轮的车碾压，结结实实地看了一个上午。我依然记得，在那个烈日当头的夏天，随着地面的热气蒸腾而上的沥青的刺鼻味道，实在不好闻，却又那么“诱人”，令人兴奋。

时代的发展，也许往往就伴随着这种不那么好闻的“味道”，比如工业化时代下的浓烟和污水，信息化时代下的焦虑和不安。身处大时代下的小小的我们，通常只是将这好闻不好闻的，统统贪婪地吸入体内了吧。

雨后初晴

发表于 2018-08-18 分类于 journal Valine：

昨天刚刚到南京，就遇上了狂风暴雨。从上午开始一直下到半夜，待在屋子里只能感受到压抑昏沉的天色，和潮湿闷热。

今天天空放晴，明晃晃的阳光仿佛挣扎着拼命穿过云层，洒向大地。站在阳光下回想，过去的一个月就像一场梦。从祖国的东南方前往中部，又飞向最西北的新疆，在短短三天里，竟又穿过了大半个中国，重新坐在熟悉的位置上，一切都不太真实。

如今，我也正站在人生的十字路口上，头顶时而狂风暴雨，时而烈日当空，而我却如身处重重迷雾中，看不清方向。

我的第一篇博客

发表于 2018-07-25 分类于 journal Valine：

经过一番安装、报错、查教程，终于用Github+Hexo似懂非懂地搭建起了一个简陋的博客。一边看着手中的博客渐渐成型完备，一边不得不惊异于前人的聪明才智。原来还可以这样简单方便地建立自己的个人网页。

写博客有点像记日记、做运动、背单词，开始时总是兴致满满，坚持不了多久就会搁置一旁。古人说行百里者半九十，其实另外一半大多数也只到十吧。希望这一次，我能够走得稍远一些。

期盼已久的假期，终于要从今天开始了。Enjoy it~
